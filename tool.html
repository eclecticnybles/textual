A Short Guide To Textual

You tell textual what to do via query parameters. Those are the bits at the end of the URL after the first ?. Query parameters look like this: key=value and are seperated with &. So, if you wanted to set the color and text parameters, it would look like this: ?color=turquoise&text=Textual.

info

Ignore all other parameters and get some information on the server

text

The text to draw itself

forceraw

Force the server to return the image instead of the HTML with the opengraph tags for embedding.

font

Set the font to write subsequent text in.

varient

What version of a font to use. Ex: regular, bold, italic

fs; fontsize

The size of the following texts in pixels

c; color

The color of the following texts. This will override pattern if it was previously set. TODO: Describe accepted colour formats. In another section.

pattern

Currently only accepts two pre-programmed pattern: trans, nonbinary

align

Sets the alignment of the following texts. Accepted values are: right, center, left

aspect

Force the image into the specified aspect ratio. Accepts floats.

bc; bcolor

Background color. If set multiple times, the last one will be taken.

bpattern

Overrides bcolor

pad

Pad all sides of the image by this number of pixels
